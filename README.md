## Run Locally

Clone the project

```bash
  git clone https://github.com/aniltulebag/search-packages.git
```

Go to the project directory

```bash
  cd search-packages
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Deployment

To deploy this project run

```bash
  npm run build
```
